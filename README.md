# 项目名称
**NetDist网盘即时通讯系统**

## 担任角色
**独立开发**

## 项目时间
2025年3月 - 2025年5月

## 项目描述

基于Qt框架开发的C/S架构即时通讯软件，实现用户注册登录、好友管理、即时聊天、云端文件存储与分享等功能。采用TCP Socket进行网络通信，支持大文件断点续传，具备完整的数据库持久化能力。

**核心功能具体实现**：

1. **网络通信架构设计**
   - 采用TCP Socket实现客户端与服务器的可靠通信
   - 设计自定义PDU（Protocol Data Unit）协议解决粘包/半包问题
   - 实现消息类型枚举，支持42种消息类型（注册、登录、好友、聊天、文件等）
   - 使用柔性数组优化协议数据结构，减少内存拷贝

2. **单例模式与线程安全**
   - Client、Server、OperateDB等核心类采用单例模式
   - 构造函数私有化，删除拷贝构造和赋值运算符
   - 确保全局唯一实例，避免资源冲突

3. **多线程并发处理**
   - 服务器使用QThreadPool线程池管理客户端连接
   - 每个客户端分配独立线程处理请求，提高并发性能
   - 客户端文件上传采用独立线程，避免阻塞UI

4. **数据库设计与操作**
   - 使用SQLite存储用户信息、好友关系
   - 设计用户表(user)和好友关系表(friend)
   - 封装OperateDB类实现数据库CRUD操作

5. **文件传输与断点续传**
   - 实现大文件分块传输机制（4KB/块）
   - 设计断点续传功能，支持网络中断后从断点继续上传
   - 客户端使用BreakpointManager持久化存储断点信息
   - 服务器端采用追加模式写入文件，确保数据完整性

6. **Qt信号槽机制**
   - 使用Qt信号槽实现对象间松耦合通信
   - 处理UI事件、网络消息、线程间通信
   - 利用Qt对象树管理内存，自动释放子对象

**技术栈**：C++11、Qt5、SQLite、TCP/IP、多线程、设计模式
